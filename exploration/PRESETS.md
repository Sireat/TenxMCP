## How to Add a New Preset

### Adding a Music Preset

1. Open `src/ai_content/presets/music.py`
2. Create a new `MusicPreset` instance:

```python
NEW_STYLE = MusicPreset(
    name="new-style",
    prompt="[Style Description] [Instruments] [Mood]",
    bpm=120,
    mood="energetic",
    tags=["tag1", "tag2"],
)
```

3. Add it to the `MUSIC_PRESETS` dictionary:

```python
MUSIC_PRESETS: dict[str, MusicPreset] = {
    preset.name: preset
    for preset in [
        JAZZ,
        BLUES,
        # ... existing presets ...
        NEW_STYLE,  # Add here
    ]
}
```

### Adding a Video Preset

1. Open `src/ai_content/presets/video.py`
2. Create a new `VideoPreset` instance:

```python
NEW_VIDEO = VideoPreset(
    name="new-video",
    prompt="Detailed scene description with camera movements and style",
    aspect_ratio="16:9",
    duration=5,
    style_keywords=["keyword1", "keyword2"],
)
```

3. Add it to the `VIDEO_PRESETS` dictionary:

```python
VIDEO_PRESETS: dict[str, VideoPreset] = {
    preset.name: preset
    for preset in [
        NATURE,
        URBAN,
        # ... existing presets ...
        NEW_VIDEO,  # Add here
    ]
}
```

### Using Presets

Presets can be used via CLI:

```bash
# Music with preset
uv run ai-content music --style jazz --provider lyria

# Video with preset
uv run ai-content video --style nature --provider veo
```

Or programmatically:

```python
from ai_content.presets import get_music_preset, get_video_preset

music_preset = get_music_preset("jazz")
video_preset = get_video_preset("nature")
```
